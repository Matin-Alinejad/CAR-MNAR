# Reproducibility

All clinical-results tables in the paper are regenerated by deterministic
scripts (fixed seeds). After `pip install -e .`, from the repository root:

| Command | Produces | Output |
|---------|----------|--------|
| `python -m carmnar.experiments.real_data_experiments_repro` | Local vs. full-graph skeleton F1 and the local-to-full ratio (paper Table 1) | `results/real_experiments_repro/real_aggregated.csv` |
| `python -m carmnar.experiments.comprehensive_metrics_experiment` | Full-graph CPDAG-aware metrics: skeleton / v-structure / orientation F1, CPDAG-SHD, SID proxy (paper Table 3) | `results/comprehensive_metrics_repro/comprehensive_metrics.csv` |
| `python -m carmnar.experiments.baseline_comparison_experiment` | SM-MVPC vs. five baselines at 30% sigmoid MNAR (paper Table 4) | `results/baseline_comparison/baseline_aggregated.csv` |
| `python -m carmnar.experiments.mechanism_comparison_experiment` | Sigmoid vs. threshold degradation profiles | `results/mechanism_comparison/mechanism_aggregated.csv` |

Each script uses `n_rep = 20` by default; the calibrator and seeds are
deterministic, so reruns reproduce the same aggregated numbers.

## Configuration

- `CARMNAR_DATA_DIR` — override the dataset directory (default `data/raw/`).
- `CARMNAR_RESULTS_DIR` — override the output directory (default `results/`).

## Ground truth

- **Clinical** reference graphs are a six-algorithm consensus CPDAG and are
  treated as a proxy; clinical numbers are "performance relative to consensus."
- **Synthetic** graphs are known by construction and are the basis for any
  absolute-accuracy statement.
